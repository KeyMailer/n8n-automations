# n8n - AI Automation Workflow

This website shows a list of n8n automations built so far for internal teams.

## Run Locally

Clone the project

```bash
  git clone https://github.com/KeyMailer/n8n-automations.git
```

Go to the project directory

```bash
  cd n8n-automations
```

Install dependencies

```bash
  npm install
```

Start the running

```bash
  npm run dev
```
